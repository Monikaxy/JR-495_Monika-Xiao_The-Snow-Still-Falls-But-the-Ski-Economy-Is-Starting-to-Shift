# JR-495_Monika-Xiao_The-Snow-Still-Falls-But-the-Ski-Economy-Is-Starting-to-Shift
This story shows how climate uncertainty is reshaping the U.S. ski economy in one season, affecting consumer behavior, industry operations, and local economies.  Built with HTML, CSS, and JavaScript, the project uses Python for data visualization, embedding charts and published on GitHub as a portfolio piece.
